# Assets Directory

Place your app assets here:

- `icon.png` - App icon (1024x1024px)
- `splash.png` - Splash screen (1242x2436px)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `favicon.png` - Web favicon (48x48px)

## Default Placeholders

The app will use Expo's default assets if these files are not provided.

## Generating Assets

You can use online tools or design software to create these assets:

1. **Icon**: Square image, 1024x1024px, PNG format
2. **Splash Screen**: Image with app logo/branding, 1242x2436px
3. **Adaptive Icon**: Square image for Android, 1024x1024px

## Asset Guidelines

- Use high-resolution images
- Keep file sizes reasonable (<1MB per file)
- Use PNG format for transparency support
- Test on different screen sizes

- https://www.canva.com (Simple design)
